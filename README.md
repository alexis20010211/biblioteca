# Biblioteca API

Este proyecto es una API desarrollada en Spring Boot para gestionar una biblioteca. Aquí se administran libros, usuarios, categorías, roles y préstamos. Todavía no tiene una base de datos conectada, pero la estructura ya está preparada para cuando se agregue.

---

##  Tecnologías usadas
- Java  
- Spring Boot  
- Spring Web  
- Spring Data JPA (sin base de datos por ahora)  
- Maven  


## ⚙️ Cómo ejecutar

1. Clonar el repositorio:
```bash
git clone https://github.com/alexis20010211/biblioteca.git
cd biblioteca


###  Ejecutar con Maven:

mvn spring-boot:run
